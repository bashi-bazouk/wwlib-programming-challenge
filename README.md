

### Installing

From the root folder:

Install electron:

	npm install -g electron-prebuilt

Install this project

	npm install


### Developing

In the `client` folder:

	webpack --watch

In the `serve` folder:

	sudo python -m SimpleHTTPServer 80

In Chrome:

	http://localhost
